# Archive Folder

This folder contains scenes and components that have been removed from the main engine but are kept for future use in separate projects.

## Scenes

The following scenes have been moved here:
- BridgeScene
- SpiderSolitaireScene
- PhysicsTetrisScene
- JellyTetrisReduxScene
- PartitionScene (and related files)
- FlipFluidSimulationScene
- SPHFluidSimulationScene
- CloudScene
- PowderScene
- ThreeDBridgeGameScene
- MarbleMazeScene
- OptimizedParticleSystem

These scenes can be used as reference or extracted into their own projects.

## Components

The following scene-specific components have been moved here:
- TetrisPhysicsComponent (used in Tetris scenes)
- CarPhysicsComponent (used in car scenes)
- CardPhysicsComponent (used in SpiderSolitaireScene)
- CardFrameComponent (used in SpiderSolitaireScene)

## Note

The core engine now focuses on:
- TestScene
- ThreeDTestScene
- FirmGuy, SoftGuy, and SpringGuy physics systems
- Core graphics and physics engine functionality
